How I worked with Responsive Design:

For this exercise, I learned how to best layout a websites semantic structure in order to later have a functional and responsive website, that works on all screen devices. In my solution, I chose to layout the navigation and header with flex. Since that makes for easier layout changes, when the screen width becomes small like on mobile devices. However, I chose to use Grid for more general layout such as card elements where there was need for positioning things in columns and rows. Since width becomes narrower on mobile devices, I chose to place the text on top of the image for the header and stack elements on top of each over in the footer. For the solutions I went with three different break points (width of: 900px, 700px and 400px)
