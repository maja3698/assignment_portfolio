How I worked with Frameworks:

For this exercise, I played around with Bootstrap as a CSS framework. Frameworks are made for making it easier for developers to layout a website using already formatted classes. I chose to layout my website in Bootstrap over other frameworks such as Pico and Tailwind since Bootstrap 4 is known for being easy to use and very popular. Bootstrap works with two different color schemes: A light theme and a dark theme. I chose the darker theme since its gentler on the eyes.
